# LINGUAPHIX branding assets

| File | Use |
|------|-----|
| `logo-mark-original.png` | Icon mark — home nav, hero, footer, favicon |
| `logo-full.png` | Horizontal wordmark + icon — **portfolio** & **policy** nav, Open Graph preview |
| `logo-mark.svg` | Optional vector fallback |

## Rules on the site

Logos use a fixed square box and `object-fit: contain` so they never stretch when the window is resized.

## Replace the master file

Overwrite `logo-mark-original.png` in this folder to update the logo everywhere. Hard-refresh the browser (Ctrl+F5) if the old image is cached.
