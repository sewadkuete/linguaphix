# Portfolio media

See **`playlists/README.md`** for how each card maps to a folder and how to add files.

Quick steps:

1. Put files in `playlists/{id}/` (e.g. `graphic-1/01-logo.png`)
2. Run `node scripts/build-portfolio-playlists.mjs`
3. Refresh `portfolio.html`

The build outputs `portfolio-playlists.json` and `js/portfolio-playlists-data.js`.
